# 🇧🇷 Português

## 📌 Sobre o Projeto

**Expense Tracker CLI** é uma aplicação de linha de comando desenvolvida em **C#** para gerenciar despesas.

O sistema permite:

* Adicionar despesas
* Listar despesas
* Remover despesas
* Ver resumo total
* Ver resumo por mês
* Armazenar dados em JSON

Este projeto foi criado para praticar conceitos importantes de desenvolvimento em C# e aplicações CLI.

---

## 🧠 Conceitos Praticados

* Programação Orientada a Objetos (POO)
* Manipulação de JSON
* LINQ
* Estrutura de pastas
* Services e Models
* Entrada de dados no console
* Tratamento de erros com TryParse
* Estrutura de CLI
* Boas práticas em C#
